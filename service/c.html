Course Objective:

This course will introduce the basic and intermediate concepts of C programming language. During this course students will learn the basics of C programming and at the later part they will be introduced to more advanced topics. Also this course will include real life examples and implementations of C programming.


Course Scope:


After completing this course students will be able to understand:


  1. System Programming (Linux, Windows) concepts.
  2. Graphics (OpenGL) programming.
  3. Implement and understand Operating System concepts.
  4. Network programming (TCP/UDP, etc protocol )
  5. Working with Databases
  6. Design and implementation of various algorithms.
